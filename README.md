## 3、主要用到的工具。

关于mqtt的详细协议内容。请参阅：
[MQTT协议中文版](https://mcxiaoke.gitbooks.io/mqtt-cn/content/)

[**萤石云开放平台Android SDK使用说明**](https://open.ys7.com/doc/zh/book/release-note/android.html)

其中包含了所有使用摄像头的详细方法。
